# MRP Assessment Tool

**Material Requirements Planning (MRP) Assessment & Optimizer**

A powerful tool to evaluate and optimize MRP parameters, helping manufacturing and supply chain teams identify issues in their current MRP setup and generate actionable improvement recommendations.

---

## ✨ Key Features

- **MRP Parameter Health Assessment** — Automatically checks Lot Size, Lead Time, Safety Stock, Reorder Point, etc.
- **Demand Forecast Accuracy Analysis** — Compares forecast vs actual consumption
- **Inventory Performance Diagnostics** — Identifies excess stock, stockout risks, and dead stock
- **Optimization Recommendations** — Provides specific MRP parameter suggestions per item
- **Multi-Scenario Simulation** — Simulate inventory cost vs service level trade-offs
- **Visualization & Reporting** — Beautiful dashboards and exportable reports (Excel / PDF)
- **ERP Agnostic** — Works with Dynamics 365 FO, SAP, Oracle, and other ERP exports
